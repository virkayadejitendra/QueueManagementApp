Queue Management App Frontend

The Day 1 plan expects an Angular frontend project. Node.js and npm are not currently available in this environment, so the Angular CLI project could not be generated here.

Once Node.js is installed, create the Angular app from the repo root with:

npx @angular/cli new frontend --routing --style=scss

Keep the Angular app focused on the MVP screens from plan.md:
- Owner Registration
- Login
- Manager Dashboard
- Customer Join
- Customer Status
- Display Screen
